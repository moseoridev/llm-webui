# WebUI for ChatKoAlpaca

## 실행 방법

- Python 3.9+ 
- Ubuntu 20.04+
- CUDA 11.6+

위 환경에서 아래와 같이 실행합니다.

```bash
pip install -r requirements.txt
gradio app.py
```

실행 후, http://localhost:7860/ 에서 사용할 수 있습니다.
